# Bonus Vault

A premium fintech landing page for **Bonus Vault** — an app by Horizon Labs LLC that helps users track and maximize bank account bonus promotions.

## What Was Built

A full-page marketing site built with React, TypeScript, Tailwind CSS, and shadcn/ui. The design is dark, modern, and premium — inspired by Linear, Stripe, and Vercel.

### Pages & Sections

- **Navbar** — Sticky glassmorphism navigation with smooth-scroll links and a mobile sheet menu
- **Hero** — Full-viewport section with animated headline, app mockup showing live bonus tracking, and key stats
- **Features** — Six-card grid highlighting Smart Tracking, Deadline Alerts, Risk Assessment, Bonus Calculator, Portfolio View, and Bank Integration
- **How It Works** — Three-step flow with step numbers, icons, and a dashed desktop connector line
- **Testimonials** — Three user reviews with star ratings and earned-amount badges
- **Contact** — Two-column layout with contact info and a validated form (react-hook-form + zod)
- **Footer** — Links grid with Product, Company, and Support columns

### Design System

- **Colors:** Dark navy/slate background (`#0F1214`), green accent (`#4ADE80`), yellow for warnings, red for risk
- **Typography:** Bricolage Grotesque for headings, DM Sans for body text
- **Animations:** CSS keyframe animations (fade-up, float, glow-green, slide-in-right)
- **Components:** shadcn/ui Form, Input, Textarea, Button, Sheet, Toast
